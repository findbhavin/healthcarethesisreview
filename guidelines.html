✏️ For Editors & Subject Matter Experts (SMEs)

All review checks, journal requirements, and decision criteria on this page are maintained in a single file: guidelines/review_guidelines.yaml in the project repository.

To update the review framework:
  1. Open guidelines/review_guidelines.yaml in any text editor
  2. Add, edit, or remove checks under the relevant stage's checks: list
  3. Update the version and last_updated fields in metadata:
  4. Add an entry to changelog: at the bottom of the file
  5. Save and commit the file — the next review will use your updated guidelines

To add a new journal, add an entry under journal_overrides: with its scope, word limits, required sections, and reference style.

See DEVELOPER_GUIDE.md in the repository for detailed step-by-step instructions.
